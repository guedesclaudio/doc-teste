# Explorador de Entidades

Pesquise e filtre as entidades da plataforma SDS por **nome** ou por **campo de inferência** (ex: `farm_uuid`, `animal_group_uuid`).

> O índice é gerado automaticamente a partir dos arquivos de documentação oficiais do repositório `ponto-b-sandbox`.
