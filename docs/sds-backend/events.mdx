# Explorador de Eventos

Pesquise e filtre os eventos da plataforma SDS por **nome** ou por **campos do payload** (ex: `farm_uuid`, `animal_uuid`).

> O índice é gerado automaticamente a partir dos arquivos de documentação oficiais do repositório `ponto-b-sandbox`.
