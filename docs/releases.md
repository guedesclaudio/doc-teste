## Política de versionamento

| Incremento | Quando usar |
|---|---|
| **Major** (X.0.0) | Mudanças que quebram a API pública ou o schema de regras |
| **Minor** (0.X.0) | Novas regras, novos grupos ou novas funcionalidades sem quebra |
| **Patch** (0.0.X) | Correções de bugs e refatorações sem alteração de API |
